# TypeScript 从 0 学习

这个仓库用来按阶段学习 TypeScript。每一课都应该包含三件事：

1. 理解一个核心概念。
2. 写一个小练习。
3. 运行类型检查，确认没有类型错误。

## 常用命令

```bash
npm install
npm run check
npm run start
```

## 学习路线

1. 基础类型、类型推断、函数类型。
2. 对象、数组、元组、字面量类型。
3. union、interface、type、类型收窄。
4. 泛型。
5. 类、访问修饰符、抽象类。
6. 模块、声明文件、第三方库类型。
7. 工程化：tsconfig、严格模式、构建和调试。
8. 实战项目：用 TypeScript 写一个命令行任务管理器。
